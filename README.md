# Tenzies React Game <sub><sup>(Scrimba React Course section 4 project)</sup></sub>
This is a solution to the Tenzies Game project on Scrimba. 

It's a tenzis game that shows you 10 dice randomly with a value between 1 - 6 of course! Once you've selected the disered dice, you can display new ones by rolling the button. The game ends once you've selected the 10 dice that have the same value. It was fun building this mini-game using React ðŸ¤©
